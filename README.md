# Transformaciones Geométricas 2D con Python

## Descripción

Este proyecto fue desarrollado como parte del tutorial final de Animación por Computadora.

El objetivo es demostrar la aplicación de las principales transformaciones geométricas bidimensionales utilizando Python y la biblioteca Matplotlib.

## Transformaciones Implementadas

### 1. Traslación

Permite desplazar una figura dentro del plano cartesiano.

Archivo:

* traslacion.py

### 2. Escalado

Permite ampliar o reducir el tamaño de una figura.

Archivo:

* escalado.py

### 3. Rotación

Permite girar una figura respecto al origen.

Archivo:

* rotacion.py

### 4. Proyecto Integrador

Aplicación de todas las transformaciones sobre una nave espacial interactiva.

Archivo:

* proyecto_final.py

## Tecnologías utilizadas

* Python 3
* NumPy
* Matplotlib
